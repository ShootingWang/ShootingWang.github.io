表格

  • Markdown表格中换行、合并单元格

  • 插入表格：

    <div>
<style scoped>
  .dataframe tbody tr th:only-of-type {
      vertical-align: middle;
  }

  .dataframe tbody tr th {
      vertical-align: top;
  }

  .dataframe thead th {
      text-align: center;
  }
</style>
<table border="1" class="dataframe">
<thead>
  <tr style="text-align: center;">
    <th>标题行</th>
    <th>第一列标题</th>
    <th>第二列标题</th>
  </tr>
</thead>
<tbody>
  <tr>
    <th>第一行</th>
    <td>$a_{11}$</td>
    <td>$a_{12}$</td>
  </tr>
  <tr>
    <th rowspan="2">合并两行单元格</th>
    <td>$a_{21}$</td>
    <td>$a_{22}$</td>
  </tr>
  <tr>
    <td>第三行就少一个单元格</td>
    <td>$a_{32}$</td>
  </tr>
  <tr>
    <th>第四行</th>
    <td colspan="2">合并两列</td>
  </tr>
  <tr>
</tbody>
</table>
</div>

效果：

标题行 第一列标题 第二列标题
第一行 $a_{11}$ $a_{12}$
合并两行单元格 $a_{21}$ $a_{22}$
第三行就少一个单元格 $a_{32}$
第四行 合并两列
