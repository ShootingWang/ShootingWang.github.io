Python中的对象包含三个基本要素：

  • id （身份标识）
  • type （数据类型）
  • vlaue （值）

is

身份运算符

比较变量的 id 地址是否相同，即是否指向同一块内存地址

==

比较运算符

比较变量的 value 是否相同，即不管是否是同一块内存地址，只要其值相同即可

## 涉及Python的深浅拷贝问题
##数字、字符串等不可变数据类型
##value一样，id一样
a = 2
b = 2

print(id(a), id(b))
140710848664416 140710848664416

print(a == b, a is b)
True True

##对于列表和字典等可变数据类型
##即使value一样，id也不一样
a = [1, 2, 3]
b = [1, 2, 3]

print(id(a), id(b))
2348645341512 2348645363144

print(a == b, a is b
True False
