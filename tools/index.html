神奇工具/网站汇总

PPT / 可视化

iSlides

  • PPT万能免费插件
  • 网站：https://islide-powerpoint.com/en/downloads

优品PPT模板

  • 在线网址：ypppt.com

iconfont

  • 阿里巴巴矢量库
  • 在线网址：iconfont.cn/?spm=a313x.7781069.1998910419.d4d0a486a

Think-Cell

  • 绘制甘特图、瀑布图、箭头与标签、百分堆积图、智能文本框、流程图等
  • 主要用于PowerPoint图表制作
  • 网站：https://www.think-cell.com/zh/

Wordle

  • 在线制作词云
  • 在线网址：http://www.wordle.net/

Color Blender

  • 在线网址：meyerweb.com/eric/tools/color-blend/#:::hex

色彩搭配表

  • 在线网址：tool.c7sky.com/webcolor/

视频/图片

PotPlayer

  • 视频播放器
  • 各种倍速选择
  • 功能超多！
  • 下载地址：https://daumpotplayer.com/download/

ScreenToGif

  • 录屏、录制动图
  • 网站：https://www.screentogif.com/

Snipaste

  • Snip & Paste
  • 截图神器
  • 功能超齐全的截图应用
  • 网站：https://www.snipaste.com/

办公 / 文本编辑

LaTeX / MacTex

  • LaTeX

MarkMap

  • 将Markdown笔记转换为思维导图
  • Markdown to Mindmaps
  • 在线网址：https://markmap.js.org/repl

Snip

  • Equation to LaTeX
  • 快速识别截图公式并输出LaTeX代码
  • 网站：https://mathpix.com/

Tables Generator

  • 在线快速生成LaTeX表格语法
  • 在线网址：https://www.tablesgenerator.com/latex_tables

Sublime

  • 轻便的文本编辑器

Xmind

  • 思维导图
  • 网站：https://www.xmind.cn

ProcessOn

  • 流程图
  • 在线网址：https://www.processon.com/

Axure

  • PM绘制原型图
  • 网站：https://www.axure.com/
  • 注册码：http://www.raedme.cn/keys/1754.html
  • 汉化包：网盘链接（提取码: wj34）

Visual Studio Code

  • 超好用的文本编辑器！
  • 网站：https://code.visualstudio.com

在线文本差异对比

  • 在线网址：jq22.com/textDifference

地图/地理

高德地图API

  • 用途：
    • 获取指定地点的坐标（经纬度）
    • 根据坐标（经纬度）获取地点信息
  • 在线网址：lbs.amap.com/tools/picker
